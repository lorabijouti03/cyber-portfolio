How to run locally
1. Clone the repository:
   git clone https://github.com/lorabijouti03/cyber-portfolio.git
2. Change into the project directory:
   cd cyber-portfolio
3. Open index.html in your browser (double-click or use a local web server):
   - Double-click index.html to open in your default browser, or
   - Use a simple HTTP server (Python):
     python3 -m http.server 8000
     then open http://localhost:8000 in your browser.
